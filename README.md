# Booking System

## Overview
This project is a booking system that allows users to make reservations and manage their bookings. It features a user-friendly interface for submitting booking requests and viewing existing bookings.

## Files
- **templates/index.html**: Contains the HTML structure for the booking system, including a form for booking and a table to display existing bookings.
- **templates/styles/styles.css**: Custom styles to enhance the appearance of the booking system.
- **static/css/styles.css**: Additional CSS for global styles or other non-template specific styles.
- **app.py**: The main application file that handles backend logic, including booking management and routing.

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd booking-system
   ```
3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```
4. Run the application:
   ```
   python app.py
   ```

## Usage
- Open your web browser and go to `http://localhost:5000` to access the booking system.
- Fill out the booking form to make a reservation.
- View and manage your bookings in the table below the form.
